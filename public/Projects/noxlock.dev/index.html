    This project is what you're staring at right now.

    My portfolio is the first HTML/CSS project I put a lot of effort into, and my first taste of using a CSS framework such as Bootstrap.

    During the time developing my Portfolio I also had to learn how to do a few things. I had never set up hosting before, which meant setting up A Records with my purchased domain, implementing Content Security Policies, integrity on resources, firebase configuration tweaking, the whole nine.


    This project was also my first taste of some workflow related things, creating, deleting, merging branches, and setting up a CI/CD pipeline with CircleCI.
